Snek 1.0 Core Command Set

Snek 1.0 is defined by its compact, hyphenated command structure, optimized for speed and simplicity on resource-constrained devices. The syntax is highly specialized for console and embedded interaction.

Command Functionality Example Syntax
anim,print "..." Displays the text character-by-character using an animated, typewriter-style effect. anim,print "hello"
anim,<var>.input "..." Presents the prompt text animated and in cyan. Assigns the user's input to <var>, with the answer box appearing on a new line prefixed by a green `>`. anim,name.input "whats your name"
wait <number> Pauses the program's execution flow for the specified amount of time (<number> time units) before proceeding to the next command. wait 500
\] Prints a single blank line to the console/output, serving as an efficient way to add vertical space without needing a dedicated 'print' function. \]
move.up/down/etc A control command that handles linear movement for the main character, cursor, or active object within the virtual environment. move.right 100

The Syntax Shift: Conditional Logic

The most dramatic difference lies in how users express logic. Comparing the same "Apple Color" logic across the three paradigms shows the shift from programmatic syntax to semantic intent.

Python (Programmatic)

if apple.color == 'red':
  continue_process()
elif apple.color == 'yellow':
  stop_process()
else:
  end_program()

Requires strict indentation, specific operators (==), and function calls.

Smart Snek (Structured)

if apple is red then continue
elif apple is yellow then dont continue
else end;

Simplified, keyword-based structure. Easier to read but still requires adhering to specific block rules.

AI Snek (Semantic)

"if apple is red then let them, but if other colors then dont let them"

Pure intent. No specific syntax. The Cloud Compiler translates "let them" to the appropriate code.

Cognitive Load Comparison

Reducing Friction

Smart Snek reduces cognitive load by removing strict syntax like brackets and complex operators.

AI Snek effectively removes syntax entirely, shifting the burden from the user's memory to the Cloud LLM's processing power.
