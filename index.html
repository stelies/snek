The Conversational Mandate

The core difference lies in the balance of Narrative Flow (snek) and Raw Power (Python). Python 3.14 sacrifices conversational readability for concise, symbolic efficiency, relying on terse operators and implied structure like indentation.

snek v2.2, by contrast, prioritizes low cognitive overhead for new users through explicit, English-like constructs (is greater than, end;). This focus makes the final v2.2 specification exceptionally easy to read, but naturally limits its performance ceiling compared to the C-based Python VM, making it ideal for scripting and educational use.

The implementation of a simple counting loop showcases the primary syntactical differences, particularly in flow control and block definition. Note the clear, explicit language of snek v2.2 versus the terse, symbolic nature of Python.

v1_goto.snek 
let count be 0
label start
if count > 5 goto finish
print count
let count = count + 1
goto start
v2_basic.snek 
let count be number 0
repeat
  print count
  set count = count + 1
until count is greater than 5
v2.2_conversational.snek 
let count be number 0
while count is less than 5 do
  print count
  set count = count + 1
end;
end;

Installation: The snek Drop-in Method

snek is designed for embedded integration. It requires a Python host environment to run, acting as a language extension rather than a standalone executable like Python 3.14. This simplified setup contributes to its high "Ease of Installation" score.

1

Prerequisite Check

Ensure a modern Python 3 interpreter (like 3.14) is installed on your system. This is the mandatory runtime host for snek scripts.

2

Embed the Interpreter

Place the core snek.py file directly into your Python project directory. No external package manager is required.

3

Execute via snek.run()

Import the file and pass your snek code as a string to the snek.run() function in your main Python script, making integration seamless.

main.py (Host)
import
snek


# Define your snek v2.2 code
snek_code = """
let names be list [string "Alice", string "Bob"]
call print_names
end;
"""

# Run the snek code within the Python host
snek.run(snek_code)
